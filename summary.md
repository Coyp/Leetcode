# 数学相关
1.最大公约数  
```python
while y % x != 0
  y, x = x, y % x
return x 
```
2.向右移动数组  
等效于对数组进行三次翻转  
1）全部翻转一次   
2）根据移动的位置左边部分翻转一次  
3）再将右边部分翻转一次  
3.快乐数  
所有不快乐数的数位平方和计算，最後都会进入 4 → 16 → 37 → 58 → 89 → 145 → 42 → 20 → 4 的循环中  
4.素数的性质  
1) 任何大于1的整数是质数或者是质数的乘积且此质数组合唯一  
可以根据此条作为map的key
